## 🧠 Key Concepts

### User Context Manager
Manages user profile, learning preferences, and session state.
- Where: `Services/UserContextManager.swift`
- What: Persistent user data + session tracking
- Use: Every time you need user info in a View

### Chat View Model
Enhanced to use context for personalization.
- Where: `ViewModels/ChatViewModel.swift`
- What: Message analysis, dynamic prompts, session end
- Use: When user sends message to mentor

### Session Context
Temporary session data (resets after session ends).
- Message count, mistakes, topic discussed
- Used to track progress within a session
